# Botainer Mini App

Frontend for Botainer Telegram Mini App.

## Tech Stack

- React 19
- TypeScript 5
- Vite 6
- Tailwind CSS 4
- Telegram WebApp SDK

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Environment

The app expects the API server to be running on `http://localhost:8080`.

Vite proxy is configured to forward `/api` requests to the backend.

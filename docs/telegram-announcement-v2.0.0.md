# Telegram Channel Announcement - v2.0.0

🎉 **Botainer v2.0.0 - Telegram Mini App Release!** 🎨

¡Gran noticia! Botainer ahora tiene una **Mini App visual** que puedes abrir directamente desde Telegram. 🚀

## ✨ Novedades

🌙 **Dashboard con tema oscuro**
- Interfaz moderna y atractiva
- Stats cards con contenedores running/stopped
- Acciones rápidas: Start, Stop, Restart
- Actualización en tiempo real

📱 **Acceso instantáneo**
1. Abre @botainerbot
2. Toca el botón "🐳 Dashboard"
3. ¡Gestiona tus contenedores visualmente!

🛠️ **Stack técnico**
- React 19 + TypeScript 5
- Vite 6 + Tailwind CSS 4
- API REST en Go
- WebSocket para updates en vivo

## 🎯 Próximamente

- 📝 Visor de logs en vivo
- 📈 Gráficos de recursos
- 🔍 Búsqueda y filtros
- 🔔 Notificaciones push
- 🐳 Gestión de Docker Compose

## 📦 Actualizar

```bash
docker pull yoniergomez/botainer:latest
docker compose up -d --build botainer
```

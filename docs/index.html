INSTALACIÓN

En marcha en 3 pasos

Solo necesitas Docker, Docker Compose y un token de Telegram.

1

Clonar el repositorio

Descarga el proyecto en tu servidor.

git clone https://github.com/YonierGomez/botainer.git
cd botainer
2

Configurar el token

Copia el archivo de ejemplo y añade tu token de @BotFather.

cp .env.example .env
# Edita .env y añade tu TELEGRAM_BOT_TOKEN
3

Iniciar el bot

Docker construye la imagen y arranca el contenedor automáticamente.

docker compose up -d --build
docker logs -f botainer   # Deberías ver: Bot iniciado: @tu_bot

CI / CD

Build multi-arch automatizado

Pipeline semanal que detecta nuevas versiones de Alpine y publica imágenes para amd64 y arm64 sin QEMU.

🔄 Docker Multi-Arch CI

El workflow se ejecuta cada lunes a las 08:00 UTC. Compara la versión de Alpine con el último release y, si hay cambios, construye en runners nativos, fusiona el manifiesto multi-arch y crea un GitHub Release con changelog.

# Fases del pipeline
1. check-version  → Detecta si Alpine cambió
2. build          → Compila en ubuntu-latest (amd64) y ubuntu-24.04-arm (arm64)
3. build-and-push → Fusiona digests en manifiesto multi-arch
4. release        → Crea GitHub Release con changelog de Alpine

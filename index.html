🚀 Instalación Rápida

1. Clonar el repositorio

git clone https://github.com/YonierGomez/botainer.git cd botainer

2. Configurar variables de entorno

cp .env.example .env nano .env

Solo necesitas el token de Telegram de @BotFather. Las notificaciones se activan automáticamente al enviar el primer mensaje al bot.

3. Iniciar el bot

docker compose up -d --build

4. ¡Listo! 🎉

Abre Telegram y busca tu bot. Envía /start para comenzar.
